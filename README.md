# HIT.PY Dev
Website for a Harare Institute of Technology python developers.


### Getting Started

1. Create and activate your virtual environment.
2. git clone https://github.com/HITpyDevs/website.git
3. pip install requirements.txt -r

### Prerequisites

* python 3.6
* sqlite3
